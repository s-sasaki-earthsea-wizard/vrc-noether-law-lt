# なぜエネルギーは保存する？ 〜自由落下でわかる"対称性"とネーターの定理〜

**YouTube動画脚本ドラフト**

## 【第1幕：問いかけ】(2-3分)

### 1. つかみ (30秒)

2つのボールを同じ高さからボールを落とすと、必ず同じ速度で地面に着きますよね。
ガリレイの時代に実証されたような、ものすごく当たり前に思える現象です。
これは位置エネルギーと運動エネルギーを足した値が必ず一定になる、
というエネルギー保存の法則から導くことができます。

でも、なぜエネルギーは保存するのでしょうか？
法則というくらいだから、そうなることが物理の「当たり前」というような気もします。
実はこの「当たり前」の裏に物理学で最も重要な考えのひとつである、「対称性」が潜んでいます。
対称性と保存則を結びつけるネーターの定理。それを一緒に探っていきましょう。

「対称性があれば、保存則がある」
...これだけだとよく分かりませんよね？
この定理の意味を今から自由落下という最もシンプルな例で解き明かしていきましょう。

物理には、不思議な「保存則」というものがたくさんあります。
これらは、「どんなに複雑な現象でも、ある量は決して変わらない」という法則です。
でも、なぜこんな法則が存在するんでしょうか？
「これはそういうことになっているから」と覚えている人も多いと思います。
でも実は保存則はもっと深い理由から必然的に生まれてくるんです。

## 【第2幕：探求と発見】(7-9分)

### 4. 自由落下の実験 (2分)

まずは実験から始めましょう。
ボールを地面から高さhの位置に置きます。

**[スローモーション再生。エネルギーバーを表示]**

落ちていく間、2種類のエネルギーが変化しています。
まず位置エネルギー。高いところにあるほど大きな位置エネルギーを持っています。式で書くと、mghです。

そして運動エネルギー。速く動いているほど大きなエネルギーです。
式で書くと、½mv²です。

落下すると位置エネルギーが減っていって運動エネルギーが増えていきます。
でもこの2つの合計はずっと一定なんです！

**[画面に式を表示]**

```
E = ½mv² + mgh = 一定
```

位置エネルギーと運動エネルギーの合計は運動の間変わらない。これがエネルギー保存則です。

地面に着く時、位置エネルギーがゼロになって全部が運動エネルギーになります。
このとき速度は√(2gh)と計算できます。

でもなぜエネルギーは保存されるんでしょうか？
ここからが今日の本題です。

### 5. 時間対称性の発見 (2-3分)

**[複数の時計を空間に出現させる]**

ここで素朴だけど重要な事実について考えてみましょう。
今日の0時にこのボールを落としてみます。
地面に着く速度は、√(2gh)です。
では同じ日の12時に落としてみましょう。
結果はもちろん同じで地面に着く時の速度は√(2gh)です！

では明日の朝6時なら？
言うまでもなく結果は同じです！

何時に実験しても結果は変わりません。
時間が変わっても結果は影響を受けない。これが時間対称性です。
「いつやっても同じ」ということを少し専門的な言葉を使うとすると「時間に対して対称的」ということができるのです。

**[画面に図解]**
```
時刻t   → 落下 → 速度 v
時刻t+1 → 落下 → 速度 v  ← 同じ！
時刻t+2 → 落下 → 速度 v  ← 同じ！
```

「そんなこと当たり前じゃん！」と思うかもしれません。
でもネーターの定理はこの時間対称性がエネルギー保存則を生み出していると教えてくれるんです。

---

### 6. 空間対称性の破れ (2分)

**[ボールを異なる高さに配置]**

時間を変えても結果は同じでした。
では高さを変えるとどうなるでしょう？

高さhから落とすと速度は√(2gh)でした。
少しだけ高い場所から落とすと地面に着くときの速度も少しだけ増えます。

あれ？高さが違うと結果が変わりますね？
つまり高さ方向には対称性がないんです。

**[地球の図を表示。重力の矢印]**

なぜか？それは地球の重力があるからです。
地球の重力は高さによって位置エネルギーを変化させます。
だから空間は鉛直方向に一様ではない。

**[対比図を表示]**

でも、ちょっと待ってください。
ボールを横方向にずらしてみましょう。

**[横にずらして落とす]**

重力は鉛直下向きに一様にかかるの横にずらしても落下の結果は変わりません。
つまり横方向には対称性があるんです。

**[まとめの図]**
```
✅ 時間をずらす  → 結果同じ → 対称性あり
✅ 横にずらす    → 結果同じ → 対称性あり
❌ 高さを変える  → 結果変わる → 対称性なし
```

**[カメラに向かって]**

重力が鉛直方向の空間対称性を破っているんです。
そして対称性がない方向には対応する保存則もありません。
逆に言えば対称性があるところに保存則が生まれるんです。
これがネーターの定理の核心です！

### 7. ネーターの定理の核心（拡張版）(4-5分)

#### 7-1. 問題提起 (30秒)

**[画面に整理]**

```
✅ 時間対称性がある（いつ落としても同じ）
✅ エネルギーが保存される
```

さてここまでで分かったことを整理しましょう。
自由落下運動はいつ落としても結果が変わらない、つまり時間対称性があります。
そして位置エネルギーと運動エネルギーの合計は一定です。
このふたつの事実がどうして結びつくんでしょうか？

これを説明してくれるのがネーターの定理なんです。
少しだけ数式を使いますがコンセプトだけ掴んでもらえるように解説していきます。

#### 7-2. ラグランジアンの登場 (1分30秒)

**[黒板風のVRボードを出現させる]**

ネーターの定理にたどり着くためにラグランジアンと呼ばれる関数を導入しましょう。
このラグランジアンには物体の運動の全情報が詰まっているんです。
導出は省略しますがラグランジアンはこのように計算されます。

```
L = T - V
```

Tは運動エネルギー、Vは位置エネルギーです。
ラグランジアンLはこの2つのエネルギーの差なんです。
なんで引き算？と思うかもしれませんが、
今日は「運動エネルギーと位置エネルギーの差を計算した、物体の運動を記述できる便利な関数」だと思ってください。
ラグランジアンLが決まれば物体がどのように動くか、すべて計算できるんです。
では自由落下のラグランジアンを見てみましょう。

```
L = T - V
  = ½mv² - mgh
```

**[VRChatで落下する物体の横にLの値を表示]**
**[アニメーション: Lの値の変化]**

ボールが落下するとき、このラグランジアンLの値が変化している様子がわかります。

#### 7-3. ラグランジアンから運動方程式へ (40秒)

**[カメラに向かって]**

でもラグランジアンと物体の運動はどう結びついてるんでしょう？
「ラグランジアンが決まれば物体がどう動くかすべて計算できる」と言いましたが、ラグランジアンと物体の運動を結びつけるのがオイラー-ラグランジュ方程式という方程式です。

**[EL方程式をサラッと表示]**

見た目はちょっと複雑ですが今日は深入りしません！
でも自由落下のラグランジアンをこの方程式に入れるとニュートンの運動方程式と同じように、実際に加速度 a = -gが実際に出てくるんです。

**[アニメーション: L → EL方程式 → 運動方程式]**

自由落下のような単純な運動だとわざわざラグランジアンとオイラー-ラグランジュ方程式を使うメリットは薄いのですが、より複雑な運動を扱うときにはこの方法が便利です。

ラグランジアンは運動の全情報が詰まっている関数で、ラグランジアンがわかればそれをオイラー-ラグランジュ方程式に代入することで機械的に運動方程式を計算できる、とだけ覚えてもらえればOKです。

ではこのラグランジアンと対称性がどう関係するのか見ていきましょう。

#### 7-4. 時間対称性の意味 (1分30秒)

時間対称性を思い出しましょう。
いつ実験しても結果は同じでしたね。

**[アニメーション: 2つのボールが時間差で落下]**

ここに2つのボールがあります。
1つ目を落として...着地しました。
では時間差をおいて2つ目のボールを同じ高さから落とします。

**[グラフに注目]**

横軸は落下を開始してからの経過時間です。
見てください、2つの曲線が完全に重なっています！

**[強調]**

つまり「いつ落としたか」は関係ない。
落下を始めてからの経過時間だけでラグランジアンは決まるんです。

ラグランジアンには時刻tが直接含まれていません。
重力gも質量mもいつ測っても同じ値だからです。
これが時間対称性の数学的な正体です。
ラグランジアンが時刻に依存しないことと、時間対称性があることは同じ意味なんです。

#### 7-5. 対称性から保存量へ (1分)

**[アニメーション: noether_energy_conservation_manim.py 前半]**

では、対称性から保存量がどう出てくるのかを見てみましょう。

**[ラグランジアンの全微分の式を表示]**

ラグランジアンを時間で微分した式を考えます。

```
dL/dt = ∂L/∂q · q̇ + ∂L/∂q̈ · q̈ + ∂L/∂t
```

**[最後の項 ∂L/∂t をハイライト]**

この最後の項に注目してください。
これは「ラグランジアンが時刻にどれだけ直接依存しているか」を表しています。

**[強調表示]**

時間対称性があるとき、つまり「いつ実験しても同じ」のとき、
この項はゼロになります。
これを計算すると...

```
d/dt(q̇ · ∂L/∂q̇ - L) = 0
```

こんな式になります。
時間微分がゼロということはカッコの中身は時間が経っても変わらない。
つまり保存量が存在するんです！

```
q̇ · ∂L/∂q̇ - L = 保存量
```

---

#### 7-6. 保存量はエネルギーだった (1分)

**[アニメーション: noether_energy_conservation_manim.py 後半]**

では、この保存量の正体を確かめましょう。
自由落下のラグランジアンを代入して計算すると...

**[ラグランジアンを表示]**

```
L = ½mq̇² - mgq
```

```
q̇ · ∂L/∂q̇ - L = ½mq̇² + mgq
```

**[T と V のラベルを表示]**

½mq̇² は運動エネルギー T、
mgq は位置エネルギー V です。

**[大きく強調表示]**

```
E = ½mv² + mgh = 一定
```

**[カメラに向かって、笑顔]**

運動エネルギーと位置エネルギーの和になっていますね。
これはまさにエネルギーです！

ここで大事なのは実験結果や物理的直感を一切使っていないことです。
ラグランジアンが時間によって変わらない、という数学的な条件だけからエネルギー保存則が導かれるんです。

対称性から保存則が必然的に導かれること、これがネーターの定理の威力なんです！

---

#### 7-7. ネーターの定理の普遍性 (1分30秒)

**[カメラに向かって]**

そしてネーターの定理が凄いのはこれがすべての対称性で成り立つと証明したことなんです。

**[運動量保存のアニメーション: videos/momentum_conservation.mp4]**

例えば、空間並進対称性を考えてみましょう。
ラグランジアンが位置xに依存しないとき、
同じ質量の2つの物体が衝突すると速度が交換されます。
でも2つの運動量の合計は常に一定なんです。
これが運動量保存則です。

```
空間並進対称性 (∂L/∂x = 0)
    ↓
運動量保存 (mv₁ + mv₂ = 一定)
```

**[角運動量保存のアニメーション: angular_momentum_conservation.mp4]**

もう一つ、回転対称性を見てみましょう。
ラグランジアンが角度θに依存しないとき、
太陽を回る地球は近いところでは速く、遠いところでは遅く動きます。
でも同じ時間で掃く面積は常に等しいんです。
これがケプラーの第2法則、つまり角運動量保存の法則です。

```
回転対称性 (∂L/∂θ = 0)
    ↓
角運動量保存 (面積速度一定)
```

**[3つの対称性を並べて表示]**

```
時間並進対称性 (∂L/∂t = 0) → エネルギー保存
空間並進対称性 (∂L/∂x = 0) → 運動量保存
回転対称性     (∂L/∂θ = 0) → 角運動量保存
```

**[カメラに向かって、笑顔]**

すべて同じ構造なんです。
ラグランジアンがある物理量に依存しなければ必ずそれに対応する保存量が存在する。
これがネーターの定理の威力なんです！
